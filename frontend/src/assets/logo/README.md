## 1. Logo with Text (sparkping_logo_text and sparkping_logo_text_dark)

Website headers and navigation bars
Email signatures
Marketing materials and presentations
Documentation and user guides
Any context where brand recognition is important
Loading screens and splash pages

## 2. Standalone Logo (sparkping_logo and sparkping_logo_dark)

App icons (desktop/mobile applications)
Social media profile pictures
Loading screens and splash pages
Anywhere you need a square/circular logo format

## 3. Favicon (sparkping_logo_full)

Browser tabs (website favicon)
Browser bookmarks
Mobile home screen icons
App store listings (as part of icon set)
Anywhere the logo appears very small (16×16 to 64×64 pixels)
