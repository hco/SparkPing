# SparkPing Home Assistant Add-on

Network latency and packet loss monitoring with beautiful visualizations.

Monitor your network infrastructure with continuous ping monitoring and 
real-time charts. Perfect for tracking Internet connectivity, local 
network devices, and remote services.

## Features

- Real-time latency monitoring
- Packet loss tracking
- Multi-target monitoring
- Beautiful D3.js charts
- Persistent data storage
- RESTful API
